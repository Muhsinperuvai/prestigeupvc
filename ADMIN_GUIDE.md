## Features

### 1. **Gallery Management**

- **Add Images:** Click "Add Image" button to add new gallery photos
- **Edit Images:** Click the pencil icon to edit image titles and locations
- **Delete Images:** Click the trash icon to remove images
- **Search:** Use the search bar to find images by title or location

**Steps to Add a Gallery Image:**

1. Upload your image to `/public/gallery/` folder (via FTP or file manager)
2. Go to Admin Dashboard → Gallery Management
3. Click "Add Image"
4. Enter image title and select location
5. The image path will be auto-populated
6. Click "Add to Gallery"
